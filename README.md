# 地区汽油价格推送

#### 快速开始

1. frok 本项目至你的仓库
2. 在仓库下配置环境变量：`/settings/secrets/actions`
   * `OIL_PRICE_PUSH_API             `推送 api 
   * `OIL_PRICE_PUSH_USER_EMAIL       `用户邮箱
   * `OIL_PRICE_PUSH_USER_PASSWORD    `邮箱安全码

3. 点击仓库下的 `actions` ，执行 `Run workflow` 
4. 查看邮箱是否收到邮件
